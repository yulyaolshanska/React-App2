# Task Manager

## Link to the website: [Click to view website](https://app-task-manager-app.vercel.app/)


### Installation

```shell
$ npm install
```

### Running the app

```shell
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

### Run client

```shell
cd client
```

```shell
npm i
npm run  start
```
